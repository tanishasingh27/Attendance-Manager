# Attendance-Manager

Student Attendance Manager

Student Attendance Manager is a responsive and student-centric frontend web application designed to help students track their subject-wise attendance, calculate attendance percentages, and determine whether they meet required attendance criteria.

The application focuses on real-world usability, clean logic, and persistent data storage, making it suitable for academic evaluation as well as technical portfolios.
